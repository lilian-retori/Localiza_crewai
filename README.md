# Projeto de Automação Web com CrewAI e Playwright

## Objetivo
Este projeto foi desenvolvido para automatizar a navegação no site de Relações com Investidores da Localiza, com a finalidade de localizar e baixar os arquivos PDF de "Divulgação de Resultados" referentes ao ano de 2026.

## Tecnologias utilizadas
- Python
- CrewAI
- Playwright
- Qwen via API compatível

## Estrutura
O projeto foi dividido em módulos para facilitar manutenção, leitura e organização do código.

## Execução
1. Ativar ambiente virtual
2. Instalar dependências
3. Executar `python main.py`

## Resultado esperado
- Navegação automatizada
- Captura de evidências por screenshot
- Download de PDFs
- Geração de relatório final
